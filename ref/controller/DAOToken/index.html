    contract DAOToken (source)

    Code deposit cost: less than 683600 gas.

    Execution cost: No bound available.

    Total deploy cost(deposit + execution): less than 683600 gas.

    DAOToken, base on zeppelin contract.

    The DAO token is a simple token that is used by DAOs to assign value to, and reward participants actions. It can be minted or burned by the DAO and also be destroyed in the event of an upgrade to a better token or in other circumstances.
